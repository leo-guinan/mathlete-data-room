Depends on the problem type:</p><p class="graf graf--p graf-after--p" id="b28c" name="b28c"><strong class="markup--strong markup--p-strong">Creation Problem:</strong></p><ul class="postList"><li class="graf graf--li graf-after--p" id="57cc" name="57cc">Collect Data until you think you have an idea on how to solve it.</li><li class="graf graf--li graf-after--li" id="4f7d" name="4f7d">Try to solve it.</li><li class="graf graf--li graf-after--li" id="903a" name="903a">Check if the solution works for random data sets as you would expect</li></ul><p class="graf graf--p graf-after--li" id="b4e1" name="b4e1"><strong class="markup--strong markup--p-strong">Optimization Problem:</strong></p><ul class="postList"><li class="graf graf--li graf-after--p" id="1186" name="1186">Pick multiple solutions to the problem.</li><li class="graf graf--li graf-after--li" id="00b5" name="00b5">Try each to solve the problem with each.</li><li class="graf graf--li graf-after--li" id="a2dc" name="a2dc">Collect data.</li><li class="graf graf--li graf-after--li" id="1290" name="1290">Look at the similarities and differences of each solution. Try pieces of each again the known solution times.</li><li class="graf graf--li graf-after--li" id="f6b9" name="f6b9">Determine the most efficient arrangement of solution pieces.</li></ul><p class="graf graf--p graf-after--li" id="045d" name="045d"><strong class="markup--strong markup--p-strong">Unknown Problem:</strong></p><ul class="postList"><li class="graf graf--li graf-after--p" id="064b" name="064b">Collect data until you can create a well-defined problem.</li><li class="graf graf--li graf-after--li graf--trailing" id="8f61" name="8f61">GOTO 1</li></ul></div></div></section>
